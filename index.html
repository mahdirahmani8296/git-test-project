شروع کنید

برای استفاده از سیستم معاملات فارکس، مراحل زیر را دنبال کنید:


📦 نصب خودکار 📖 راهنمای دستی 🎯 اجرای دمو

مراحل نصب:

  1. Python 3.8 یا بالاتر نصب کنید
  2. اسکریپت نصب را اجرا کنید: python install.py
  3. TA-Lib را نصب کنید (در صورت نیاز)
  4. سیستم را تست کنید: python demo.py

نصب دستی:

  1. pip install -r requirements.txt
  2. TA-Lib را نصب کنید
  3. python forex_trading_system.py

اجرای دمو:

  1. python demo.py - اجرای بک تست
  2. python trading_dashboard.py - داشبورد
  3. python strategy_optimizer.py - بهینه‌سازی

⚠️ هشدار مهم

این نرم‌افزار صرفاً برای اهداف آموزشی و تحقیقاتی طراحی شده است. معاملات فارکس دارای ریسک بالایی است و امکان از دست دادن سرمایه وجود دارد. قبل از استفاده در معاملات واقعی، حتماً تست کامل انجام دهید.
